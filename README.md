# BioPad
A 12 key macropad with a oled display and 5 LEDS uses KMK firmware. It is supposed to be a used during work to 

## Features
- 128x32 OLED Display
- KMK
- 12 Keys
- 4 Layers Currently
- 5 LEDS

## Cad Model
It fits with 4 M3 Bolts and it has 4 M1 Bolts to hold the pcb down. 
It has 3 parts the bottem the layer blocking the keys and top.

## Firmware
I am using KMK to be able to hotswitch it to whatever people need.
There is currently 3 layers Apps Vscode and Zoom. Each of those has hotkeys related to each app. The Apps layer types in Windows + R and adds the name of a website or app.
